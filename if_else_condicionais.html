Operadores Condicionais

Também conhecidos como operadores relacionais para formar as expresões condicionais para o comando if/else

Operadores:

Igual (==)


Idêntico (===)


Diferente (!=)


Não idênticos (!==)


Menor (<)


Maior (>)


Menor igual (<=)


Maior Igual (>=)

Interpretação do if/else:

if (Condição) {
document.write('Entrou dentro do bloco if (Verdadeiro)' + 'br')
} else{
document.write('Entrou dentro do bloco do else (falso)' + 'br')
}

Observações: estamos trabalhando com condições estáticas. Diferente do dia-a-dia, onde as condições são totalmente dinâmicas
