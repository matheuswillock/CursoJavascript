Switch

O que é o comando Switch?

O comando Switch é uma condicional que visa com base em uma determinada condição seguir um determinado fluxo dentro da aplicação. (Parecido com if/else).

Diferenças entre o Swicth e o if/else.

O comando Switch é bem parecido com o if/else, mas ele nos auxiliano enetendimento

Sintaxe

switch (parâmetro) {

case 1:
// Se o parâmetro === 1
break

case 2:
// Se o parâmetro === 2
break

default:
// Se o parâmetro não encontrado
}
