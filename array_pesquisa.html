Método de pesquisa em Array

Pesquisamos elementos dentro de uma Array utilizando o método nativo .indexOf().
Seo elemento pesquisado não for contido na Array, retorna a posição do index da Array -1, e se, for positivo retorna a posição do indíce da Array onde o elemento está contido.

  • listaFilmes[0]'...'
  • listaFilmes[1]'...'
  • listaFilmes[2]'...'
  • listaFilmes[3]'...'

O exemplo abaixo vai explicar;

var listaAnimes = Array('Naruto', 'One Piece', 'Sword art online', 'My Hero Academia', 'Black Clover')

var anime = listaAnimes.indexOf('Naruto')

if (anime === -1) {
document.write('Não assistido')
} else {
document.write('Assistido e está na posição ' + anime)
}
