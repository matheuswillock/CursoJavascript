Funções - Introdução

Uma função tem o objetivo de encapsular um determinado bloco de códigoscom umobjetivo definido.
E através da chamada das nossas funções nós podemos recuperar essas instruções para utiliza-las em diversas partes das nossas aplicações.

Sintaxe

function nome(parâmetro/argumentos) {
var nome = parâmetro, parâmetro
return nome
}

Por exemplo:

function calcularAreaTerreno(terreno, comprimento) {
