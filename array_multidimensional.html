Array Multidimensional

O que são?

Arrays Multidimensional ou Bidimensional são, Arrays com arrays encadeadas dentro delas.
Podemos utilizar essa estrutura para emissão de relatórios que, retornam os dados do banco de dados e agrupam todos os dados em uma única Array.

Métodos de inclusão e exclusão de elementos

var lista_frutas = Array('Banana', 'Maçã')

lista de frutas:

  1. Banana
  2. Maçã

  • Para icluirmos um elemento ao final da nossa Array, passamos o método nativo push.

  • Incluir um elemento na primeira posição da Array, unshift.

  • Excluindo um elemento ao final do Array independete do indíce e do nome do elemento; pop.

  • Excluindo o primeiro elemento da Array; shift.

  • Incluido um elemento ao final da Array multidimensional.

  • Incluindo ao inicio da Array Bidimensional.

  • Removendo elemento do final da Array Bidimensional.

  • Removendo elemento do inicio da Array Bidimensional.
