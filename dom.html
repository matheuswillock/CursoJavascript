Dom - Introdução
Document Object Model

O que é?

API que nos permite acessar via Javascript os elementos html da página.
É através do DOM que vamos acessar e modificar os elementos HTML. Cada elemento contido no DOM é chamado de node.

Manipulando valores de inputs (texto)

Objetivo: selecionar os elementos do tipo text a apartir do objeto 'document' utilizando alguns dos métodos disponiveis na API e posteriormente afetar os valores.

Desafio:

  • Criar tres campos:
    1. Entrada de caracteres.
    2. Entrada de letras somente.
    3. Entrada de números somente.

O primeiro campo servirá para entrada de caracteres e então, com base no caracterer digitado no campo, vamos disparar uma lógica a partir de evento. Se o caracter for uma letra, vamos acrescentar o caracter ao input do tipo text. Caso contrário se o caracter digitado for um número, vamos acrescentar o caracter ao input text do tipo number.
A partir de um evento, vamos disparar uma ação e dentro desta ação, vamos precisar selecionar os elementos a partir da API do DOM para aplicar a lógica necessária.

Manipulando estilos com DOM

Essa éuma tecnica muito utilizada para fazermos com que a página reaja visualmente as ações do usuário.

Exemplos:

  • Exibir ou fechar uma tabela atraés de um botão.
  • Expandir ou retrair um menu.
A partir damanipulação visual dos elementos podemos fornecer para o usuário informações muito ricas e dar mais dinâmica as nossas aplicações.

Objetivo

Vamos criar uma página com três botões e ao clicar no botão nós vamos afetar o background de uma div.

Manipulando classes de elementos

Ao selecionar um elemento através do método 'ClassName' vamos aplicar uma classe ao elemento.
Obs: o atributo 'getElementById' mofigica o stilo de forma inline, enquanto o atributo 'getElementsByClassName' atribui uma class ao elemento.
