Eventos - Introdução

O que são?

Os eventos nos permitem capturar ações relizadas pelo usuário dentro do broswer. Desde um determinado movimento no mouse sobre algum elemento da página, um click do teclado,processos de loading, redimensionamento da tela e ações dentro de formulários.
Todos esses eventos e muitos outros podem ser combinados com funções e a patir disso podemos disparar a realização de uma lógica, combinada a uma interação do usuário com a página e até mesmo sobre a manipulação do broswer.

Eventos capturados a partir do Mouse:

  • onclick

    • Quando o elemento é clickado.

    Click aqui!

  • ondblclick

    • Quando o elemento é clickado duas vezes.

    Click aqui!

  • onmouseup

    • Quando o click sobre determinado elemento é liberado.

    Click aqui!

  • onmouseover

    • Acionado quando o cursor do moouse sobrepõe a região do elemento dentro do broswer

    Click aqui!

  • onmouseout

    • Quando o cursor sai sobre o elemento da página.

    Click aqui!

Eventos capturados a partir do Teclado:

  • onkeydown

    • Acionado no momento emque qualquer tecla é pressionada.

  • onkeypress

    • Acionado em que a tecla é liberada. Só funciona com teclas que representam uma saída de um caractere, ou seja, não funciona com o 'ctrl', por exemplo.

  • onkeyup

    • Acionado no momento em que a tecla é pressionada.

Eventos disparados a partir do Frame

  • onresize

    • Acionado quando a página é redimensionada.
    OBS: Para utiizarmos esse evento precisamos aplicar no elemento html que sofre redimensionamento de tela, logo entendemos que é o elemento body.
    Vamos aplicar o evento ao elemento do body e redimensionar a tela para que o evento seja disparado.

  • onscroll

    • Acionado quando o scroll é realizado na página. OBS: para verificar a ação deste evento, utilize o devtools - console.

Eventos capturados a partir de Formulários:

Todos os eventos serão recuperados pelo console, para evitar que ocorra loops.

  • onfocus

    • Acionado quando o elemento recebe o focu do mouse.

  • onblur

    • Acioando quando o elemento perde o focu

  • onchange

    • Acionado quando o estado é modificado. Muito comum com elementos do type select.
