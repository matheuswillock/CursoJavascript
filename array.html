Array - básico

O que é

São listas ordenadas. São variáveis que nos permitem relacionar itens associados a indíces, esses indíces são chaves.
Esse tipo de variáel tenta imitar as listas do mundo real.

Listas de frutas

  1. Banana
  2. Maçã
  3. Morango
  4. Uva

Com variáveis:

  • var fruta_1 = 'Banana'
  • var fruta_2 = 'Maçã'
  • var fruta_3 = 'Morango'
  • var fruta_4 = 'Uva'

Com um único Array:

var lista_frutas = Array();

  • lista_fruta[1] = 'Banana'
  • lista_fruta[2] = 'Maçã'
  • lista_fruta[3] = 'Morango'
  • lista_fruta[4] = 'Uva'

Uma Array serve para armazenar qualquer item fde forma relacional.
Por exemplo; uma lista funcionários, lista de compras, lista de nomes de alunos e etc. Tudo o que puder ser relacionável é possível ser incluido em arrays.
Assim, temos a vantagem de recuperar os resultados chamando o nome de uma única variável e indicando a posição do elemento.

Para criarmos uma Array, precisamos informar que uma variável recebe um objeto do tio array.

var lista_frutas = Array();


Depois, podemos adcionaros itense indicar os indíces de cada elemento.
  • lista_fruta[1] = 'Banana'
  • lista_fruta[2] = 'Maçã'
  • lista_fruta[3] = 'Morango'
  • lista_fruta[4] = 'Uva'
