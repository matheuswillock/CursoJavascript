Funções nativas para manipular strings

Valores primitivos como strings, não tem propriedades por si só. Porém,quando são interpretados pelo JS, eles são convertidos emobjetos e, com isso, são agregados propriedades e métodos pré-estabelecidos da linguagem.
Fazemmos as manipulações através de atributos, propriedades e métodos.

Os métodos e propriedades podem ser consultados no site w3schools.com

Propriedade Length

Nos retorna a quantidade de caracteres de uma string.
Por exemplo; quantos caracteres tem o meu nome? Matheus Willock.

Propriedade charAt

Retorna uma caracter com base na posição domesmo dentro da cadeia de caracteres.

Método indexOf

Retorna a possição da primeira ocorrência de um valor especifico dentro da string. Nesse caso, a ideia é encaminhar um caracter específico, então indexOf vai retornar a posição daquele caracter na primeira ocorrência,ou seja o index referente aquele caracter.
Por exemplo; se o seu nome possui a letra 'A' ele vai nos passar a primeira posição da ocorrência.

Se não houver a letra 'A' ele vai nos retornar o número '-1'.

Método replace

Ele permite,com base em um valor de pesquisa e também com outro valor de identificação, substituir textos dentro de uma string.
Por exemplo, de um refresh na página e coloque o seu sobrenome como 'Silva', o texto será substituido por 'Chica'.

Método substr

Este método permiti extrair parte de uma string seguida de um valor com base numa posição inicial seguida de uma valor que representa a quantidade de caracter a frente da posição inicial. É como se ele cortasse parte da string.
Quando indicarmos a posição inicial,devemos contar apartir do mesmo as quantidades de caracteres que desejamos. Então ele vai nos recuperar aquela parte da string recortada.

Método toLowerCase

Converte todos os caracteres da string em caixa baixa.

Método toUpperCase

Converte todos os caracteres da string em caixa alta.

Método trim

Ele remove os espaços em branco das extremidades das strings.
Repareno exemplo abaixo:
nome = ' Matheus Willock '
document.write('-' + nome.trim() + '-')
