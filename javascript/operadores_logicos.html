Operadores Lógicos

Operadores Nos permitem conectar com os teste de comparação.
Cada cmoparação resulta em true / false, com isso podemos juntar as comparações com os operadores lógicos para obter uma segunda resposta.
Isso significa que, com os operadores lógicos podemos dar mais dinâmicas aos nossos testes de condicionais. Porque se torna possível criar diversas camadas de possibilidadesno mesmo teste.
