Variáveis

Variável do tipo string:

var texto = 'Curso de JavaScript!'

Variável do tipo number:

var numeroInteiro = -7
var numeroFracionado = 10.50 (Sempre separamos os numeors fracionados com '.')

Variávei do tipo boolean:

var teste = true (true -> Verdadeiro / false -> Falso)
